# Language Roles

- **Ballerina**: Integration and API layer. Handles HTTP, orchestrates services.
- **Gleam**: Concurrent workers, FFI to Mojo, Vale, Bosque.
- **Mojo**: Performance-critical proving engine, uses Bend for GPU.
- **Bend**: GPU parallelism for MSM/FFT.
- **Zig**: Low-level cryptography with C FFI.
- **Vale**: Memory-safe PII handling.
- **Bosque**: Deterministic policy language.
- **V**: Client SDK for simplicity.
- **ReScript**: Type-safe frontend.
- **Rust**: Solana smart contract.
